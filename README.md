# Pact Smart Contracts Catalog

Repository purpose
------------------
This repository is the canonical catalog of audited and community-maintained Pact smart contracts for the Pact Community Organization. It provides a searchable index, metadata, audit status, and deployment examples for each contract.

Mission & Vision
-----------------
Mission: Make it easy and safe for businesses to start building with Pact.

Vision: A trusted Pact ecosystem where businesses can confidently start using audited, reliable open source Pact contracts.

See `ARCHITECTURE.md` for the repository layout and contribution guidelines in `CONTRIBUTING.md`.
